## 🚀 **Deployment Guide**

### **GitHub Pages (Free)**
```bash
git init
git add .
git commit -m "Deploy minimal portfolio"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/moe-portfolio.git
git push -u origin main
# Enable Pages in Settings → Pages → Main branch
```

### **Netlify (Recommended)**
```bash
# Option 1: Direct upload
# Drag entire folder to netlify.com/drop

# Option 2: CLI
npm install -g netlify-cli
netlify deploy --prod --dir=.
```

### **Vercel**
```bash
npm install -g vercel
vercel --prod
```

### **Custom Domain with HTTPS**
```
1. Purchase domain (Namecheap, GoDaddy, etc.)
2. Point nameservers to Netlify/Vercel
3. Enable auto HTTPS in platform settings
4. Set up email forwarding (optional)
```
